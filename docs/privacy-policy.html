Privacy Policy

Last updated: July 10, 2026

Trainers Web Plus is a browser extension that enhances the Pokémon TCG Asia website for registered players. This policy explains what data the extension accesses and stores.

Data We Access

The extension reads page content from the Pokémon TCG Asia website (asia.pokemon-tcg.com) while you are browsing it. This includes event listings and registration statuses visible to your logged-in account.

Data We Store

The extension stores the following data locally on your device using the browser's built-in chrome.storage API. This data never leaves your device.

  • User ID — used solely to detect if you have switched accounts, so cached data is not shown to the wrong account.
  • Event registration data — cached results of your event registration statuses to avoid repeated lookups.

Data We Do Not Collect

  • We do not transmit any data to external servers. The extension has no backend and makes no network requests of its own.
  • We do not collect passwords, credentials, or payment information.
  • We do not share any data with third parties.
  • We do not use analytics or tracking of any kind.

Permissions

The extension requests only the permissions necessary to function on the Pokémon TCG Asia website.

Changes to This Policy

If this policy changes in a future update, the updated date at the top of this page will reflect that.
